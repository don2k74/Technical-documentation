Introduction

freeCodeCamp is a free online learning environment. They offer detailed curriculums in many different coding languages. The courses are well structured to making it easy to learn. There are projects in each course that challenge you.

freeCodeCamp provides helpful resources if you get stuck. The projects in each course can be use in your portfolio. You can link your GitHub, website, LinkedIn and Twitter.

  • You can post your problem to the forum if you get stuck.
  • They have a YouTube channel with tutorials if you need help.
  • The YouTube channel also has tutorials that are not in the courses.
Html CSS

A few of the courses you will learn and use Html and CSS. Responsive Web Design course is a start if you are a beginner in Html and CSS.

You will learn to create many useful projects for your coding career.

  • Projects like personal portfolio, webpage survey form and others.
  • You can style your page in CSS like this
    • header { color: black;} header { margin: 10px;} #navbar {position: fixed;} body { background-color: #ffffff;} #navbar { min-width: 290px;}
